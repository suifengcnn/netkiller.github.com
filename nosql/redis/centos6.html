1.2. CentOS 6

安装fedora的YUM源，

				rpm -Uvh http://dl.fedoraproject.org/pub/epel/6/x86_64/epel-release-6-7.noarch.rpm

安装redis

				# yum install redis

				# chkconfig redis on

				# service redis start

备份配置文件，

				# cp /etc/redis.conf /etc/redis.conf.original

1.2.1. 主从同步

主从同步配置非常简单，只需在从服务器 /etc/redis.conf 文件中开启 slaveof 即可

					slaveof 192.168.2.1 6379
