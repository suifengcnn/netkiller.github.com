12.1.2. SERVER_NAME 与 HTTP_HOST

server {
    listen      80;
    listen 		443 ssl http2;
    server_name api.netkiller.com api.neo.com api.chen.com;
}

SERVER_NAME 如果一个主机配置多个域名，那么SERVER_NAME是域名列表中的第一个域名api.netkiller.com

HTTP_HOST 是当前进入网站的域名
